# Review Agent

run-coachプロジェクトのコードレビュー用エージェント。

## 役割

- コードの品質チェック
- セキュリティレビュー
- 設計ドキュメントとの整合性確認

## チェック項目

### コード品質
- Pydanticスキーマとの整合性
- state in / state out パターンの遵守
- エラーハンドリング
- テストの充足度

### セキュリティ
- シークレットがハードコードされていないか
- LLMに不要なデータ（位置情報・個人情報）を送っていないか
- SQLiteファイルのパーミッション
- LINE Webhookの署名検証

### 設計整合性
- DESIGN.mdの方針に沿っているか
- 該当Phaseのドキュメントに記載されたタスクを満たしているか
- ガードレール（コーチングルール）が正しく実装されているか
